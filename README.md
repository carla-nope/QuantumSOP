# QuantumSOP

QuantumSOP is a public-sector-focused AI governance, quantum readiness, and compliance documentation brand.

This starter repository contains a Vercel-ready React/Vite site with:

- Main landing page
- Glossary
- FAQ
- Compliance Vault preview
- AI governance, quantum security, and procurement intelligence sections
- SEO metadata

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This project is intended to deploy on Vercel. Import the GitHub repository into Vercel and use the default Vite build settings:

- Build command: `npm run build`
- Output directory: `dist`

## Content direction

The current site is a Phase 1 foundation. Next content additions should include:

1. A real lead magnet or waitlist form integration.
2. Full product pages for the Compliance Vault, AI Governance Starter Kit, and Quantum Readiness Kit.
3. Blog/intelligence briefing archive.
4. Case study pages for public-sector AI implementation examples.
5. Legal disclaimers and privacy policy before public launch.
